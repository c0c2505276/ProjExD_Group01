# 大富豪

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ローカルルールが搭載された 1 on 1 の大富豪
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 場に出ているカードより大きい数字を出しましょう。
* ローカルルールが搭載されている状況で勝ちましょう。

## ゲームの実装
### 共通基本機能
* トランプゲームの大富豪

### 分担追加機能
* 革命
* 8切り C0C25052/中島瑞稀
* 階段
* マーク縛り
